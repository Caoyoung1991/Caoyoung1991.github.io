1. 装饰器

装饰器

装饰器初步介绍

def now():
    print('2015-3-25')

动态增加now函数的功能，而不改变原函数。

def log(func):
    def wrapper(*args, **kw):
        print('call %s():' %func.__name__)
    return func(*args, **kw)
return wrapper

log函数里定义了wrapper函数,wrapper函数做了两件事：1.打印func函数信息，2.调用func函数。

执行

now = log(now)

就会打印日志并且输出日期。

但是一般不这样做，而是用下面的方法：

@log
def now():
    print('2015-3-25')

使用python的@语法，将装饰器置于要执行函数的顶部。

##
装饰器参数传入

def log(text):
    def decorator(func):
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

这样三层调用，打印结果如下：

execute now():
2015-3-25

##
原函数的属性

>>>now.__name__
'wrapper'

原函数的属性改变了，我们需要把其他的属性复制过来。python内置函数可以解决这件事。functools.wraps。

import functools

def log(func):
    @functools.wraps(func)
    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper
